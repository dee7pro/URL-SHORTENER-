A modern URL shortener web application built using Flask and SQLite.  
It allows users to shorten long URLs, store them, and view their history through a custom purple-black themed interface.

## Features
- URL shortening with validation
- Persistent storage using SQLite
- URL history tracking
- Copy-to-clipboard functionality
- Custom dark-themed UI

## Tech Stack
- Python (Flask)
- SQLAlchemy
- SQLite
- HTML, CSS
